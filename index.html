Poligon Web: OWASP Top 10 w Praktyce

O projekcie

Witaj w laboratorium poświęconym bezpieczeństwu aplikacji internetowych! Ten projekt to "poligon doświadczalny", który pozwoli Ci poznać najpopularniejsze podatności oraz nauczyć się, jak przed nimi chronić swoje aplikacje. Moduły interaktywne (PHP/MySQL) demonstrują różnice między kodem podatnym a bezpiecznym.

Wymagania

  • Docker & Docker Compose
  • Terminal (Linux/macOS)
  • PowerShell (Windows)

Dostęp do usług

  • Aplikacja: localhost:8080
  • phpMyAdmin: localhost:8081
  • Baza (Host): db (root/root_password)

Instalacja i uruchomienie

1. Uruchomienie kontenerów

docker compose up -d --build

2. Nadanie uprawnień (Linux/macOS)

chmod 777 src/uploads

Przygotowanie bazy danych 

CREATE TABLE IF NOT EXISTS users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100) NOT NULL,
    secret_info VARCHAR(100) NOT NULL

Zarządzanie środowiskiem

Uruchomienie: docker compose up
Zatrzymanie: docker compose down
Pełne czyszczenie:
docker compose down --volumes
Przebudowanie: docker compose up --build
